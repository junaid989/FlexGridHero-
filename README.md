# FlexGrid Hero+ (Cyber Neon)

Interactive terminal game to learn **CSS Flexbox**, **Nested Flexbox**, and **CSS Grid**.
Credits: Created by @muhjunaid_123 & @sys_adminshadow

## Run
```bash
python3 src/main.py
# or
python src/main.py
```

Works on Ubuntu, Termux, and Windows (Python 3). No external packages required.

## Features
- Lessons: Flexbox, Nested Flexbox, Grid, Advanced Grid, Dev tools
- ASCII visualizer (previews + build animation)
- Badge system and progress saving (`data/progress.json`)
- Mini code-editor challenge to type small CSS snippets
- Cyber neon intro animation (scan + typewriter)
